GRIFLOR DECO — Demo Argentina
==============================

CÓMO SUBIR A UN HOSTING
------------------------
1. Subí la carpeta "griflor-argentina-demo" a tu hosting
2. El archivo principal es index.html
3. No necesita base de datos ni backend — es 100% estático
4. Funciona en cualquier hosting: Netlify, Vercel, cPanel, etc.

HOSTING RÁPIDO GRATIS PARA DEMO (recomendado)
----------------------------------------------
• Netlify Drop: drag & drop de la carpeta en https://app.netlify.com/drop
  → En 30 segundos tenés una URL pública tipo https://griflor-demo.netlify.app

• Vercel: subí la carpeta en https://vercel.com/new

FUNCIONALIDADES DE LA DEMO
---------------------------
✓ Cursor custom con anillo y label
✓ Scroll con contador de porcentaje en nav
✓ Hero con parallax y animación de entrada
✓ Strip horizontal de productos (hover expande, drag para scrollear)
✓ Grilla de 6 productos con hover effects
✓ Modal de producto animado (clic en cualquier planta)
✓ Menú overlay con animación
✓ Scroll reveal en todas las secciones
✓ Sección About con números
✓ Trust bar y testimonios
✓ Footer completo
✓ Responsive mobile
